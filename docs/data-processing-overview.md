# Data Processing Overview

## Purpose
The platform processes user data to provide chat, RAG retrieval, account management, billing, safety, and analytics.

## Categories
- Identity and authentication data.
- Conversation and feedback data.
- Usage, cost, and quality telemetry.
- Admin action logs and security events.

## Controls
- Sanitization before analytics storage.
- Role-based access control.
- Retention jobs for old data.
- Privacy APIs for export, deletion, and memory reset.
